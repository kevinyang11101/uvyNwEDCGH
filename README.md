# 小区物业管理系统 java839

## 项目概述

小区物业管理系统是一款基于B/S模式和MVC框架的前后端分离管理系统。该系统结合了SpringBoot、Vue等现代技术，致力于提高物业管理的效率与质量。

## 技术栈

- 开发环境：IDEA/Eclipse
- JDK版本：1.8
- 数据库：Mysql
- 项目构建：Maven
- 前端运行环境：Node.js
- 核心技术：SpringBoot、Java、Vue、Mybatis-Plus

## 功能模块

### 用户功能模块

- 1.1 个人信息管理
- 1.2 费用信息管理
- 1.3 楼房信息管理
- 1.4 报修信息管理

### 管理员功能模块

- 2.1 登录与用户注册
- 2.2 用户管理
- 2.3 员工管理
- 2.4 费用信息管理
- 2.5 楼房信息管理
- 2.6 报修信息管理
- 2.7 车位信息管理
- 2.8 投诉编号管理

### 员工功能模块

- 3.1 个人信息管理
- 3.2 楼房信息管理
- 3.3 报修信息管理
- 3.4 投诉编号管理

## 系统角色

系统定义了三种角色：管理员、用户、员工，各角色权限分明，职责明确。

## 运行环境

- 服务器：任何支持Java和Mysql的服务器环境
- 客户端：现代浏览器
